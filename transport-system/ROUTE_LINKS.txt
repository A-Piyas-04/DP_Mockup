NOTES
===============================================
1. The root path "/" automatically redirects to "/dashboard"
2. All routes except "/login" are wrapped in the Layout component (includes navigation)
3. Dynamic routes require an ID parameter (e.g., /vehicles/1, /drivers/2)
4. The dev server is currently running on port 5173 (default Vite port)
5. If the port is different, check your terminal output for the actual URL
